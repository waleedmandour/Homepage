iAWE - Support & Contact

iAWE (Intelligent Academic Writing Evaluation Assistant) is an AI-assisted academic writing evaluation system built on the EASE Framework 1.0 and distributed as a Google Sheets Add-on. If you encounter technical issues with the Add-on, require assistance with API configuration (Gemini 2.5, Semantic Scholar), or have academic suggestions for the tool, please review the troubleshooting notes below and contact the developer.

Common Support Topics

  • Installation issues: Ensure you are installing iAWE from the official Google Workspace Marketplace listing, and that your domain administrator permits third-party Add-ons with the drive.file and spreadsheets.currentonly scopes.
  • API configuration: Users must supply their own API keys for Gemini 2.5 (Google AI Studio) and Semantic Scholar where required. See the README section of the project page for setup instructions.
  • Evaluation results interpretation: iAWE outputs structured EASE metrics including NAWL lemma density, Type-Token Ratio, and JSON-anchored feedback scores. Refer to the Research Transparency Notice on the project page for methodological context.
  • RAG / DDL methodology questions: The EASE Framework triangulates DBR, DDL, and RAG principles. Academic inquiries about the underlying methodology are welcome.
