AWE Desktop: Offline AI Essay Assessment

تقييم الكتابة الآلي: تقييم مقالات بدون إنترنت بالذكاء الاصطناعي

100% Local • No Internet Required • Full Privacy
محلي بالكامل • لا يحتاج إنترنت • خصوصية تامة

A native Windows desktop application that runs entirely locally with offline support. Powered by local LLMs (Ollama, LM Studio) and offline OCR engines for complete data privacy and zero internet dependency after initial setup.

تطبيق ويندوز أصلي يعمل بالكامل محلياً مع دعم العمل بدون اتصال. مدعوم بنماذج LLM محلية (Ollama, LM Studio) ومحركات OCR بدون إنترنت لخصوصية بيانات تامة وعدم الاعتماد على الإنترنت بعد الإعداد الأولي.

Windows 10/11 ويندوز 10/11 Free & Open Source مجاني ومفتوح المصدر

AWE Desktop — Track 2: Windows Desktop Application

تقييم الكتابة — المسار الثاني: تطبيق ويندوز للحاسوب

Part of the AWE System (Automated Writing Evaluation) — a Multimodal, LLM-based system for formative assessment designed for Foundation Year and Post-Foundation students at Sultan Qaboos University.

جزء من نظام تقييم الكتابة الآلي (AWE) — نظام متعدد الوسائط قائم على LLM للتقييم التكويني صُمم لطلاب السنة التأسيسية وما بعد التأسيس بجامعة السلطان قابوس.

Developed by: Dr. Waleed Mandour (2025-2026)

تطوير: د. وليد مندور (2025-2026)

Key Features

الميزات الرئيسية

Offline Operation

العمل بدون اتصال

Runs entirely on your local machine with zero internet dependency after initial setup. All AI processing, OCR, and data storage happen locally on your computer.

يعمل بالكامل على جهازك المحلي بدون أي اعتماد على الإنترنت بعد الإعداد الأولي. جميع عمليات الذكاء الاصطناعي والتعرف الضوئي وتخزين البيانات تتم محلياً على حاسوبك.

Local Onlyمحلي فقط SQLite

Local LLM Support

دعم نماذج LLM المحلية

Compatible with Ollama, LM Studio, and custom LLM endpoints. Choose from a range of models based on your hardware — from lightweight 1B models to powerful 70B models for professional-grade assessment.

متوافق مع Ollama وLM Studio ونقاط نهاية LLM مخصصة. اختر من مجموعة نماذج حسب إمكانيات جهازك — من نماذج خفيفة 1B إلى نماذج قوية 70B للتقييم الاحترافي.

Ollama LM Studio Custom API

Multiple OCR Engines

محركات OCR متعددة

Supports Tesseract, PaddleOCR, and EasyOCR for handwriting recognition. Choose the engine that works best for your handwriting style, or use Vision LLM (LLaVA, MiniCPM-V) for enhanced recognition accuracy.

يدعم Tesseract وPaddleOCR وEasyOCR للتعرف على الخط اليدوي. اختر المحرك الأنسب لنمط خطك، أو استخدم LLM بصري (LLaVA, MiniCPM-V) لدقة تعرف محسّنة.

Tesseract PaddleOCR EasyOCR Vision LLM

Privacy & Security

الخصوصية والأمان

Your essays never leave your computer. All data is stored locally in an encrypted SQLite database. No data is sent to external servers. Ideal for institutions with strict data privacy requirements.

مقالاتك لا تغادر حاسوبك أبداً. جميع البيانات تُخزّن محلياً في قاعدة بيانات SQLite مشفّرة. لا تُرسل أي بيانات إلى خوادم خارجية. مثالي للمؤسسات ذات متطلبات خصوصية صارمة.

Encrypted Storageتخزين مشفّر Zero Cloudبدون سحابة

System Requirements Checker

فاحص متطلبات النظام

Built-in tool that scans your hardware and recommends the best LLM model for your system. Checks RAM, GPU VRAM, storage space, and CPU to ensure optimal performance before installation.

أداة مدمجة تفحص عتادك وتوصي بأفضل نموذج LLM لنظامك. تتحقق من الذاكرة وVRAM ومساحة التخزين والمعالج لضمان أداء مثالي قبل التثبيت.

Hardware Scanفحص العتاد Auto Recommendationتوصية تلقائية

Microsoft Store Ready

جاهز لمتجر مايكروسوفت

Available as NSIS, MSI, and MSIX installer packages. Can be distributed through the Microsoft Store for easy enterprise deployment. Supports auto-update for seamless maintenance.

متوفر كحزم تثبيت NSIS وMSI وMSIX. يمكن توزيعه عبر متجر مايكروسوفت لنشر سهل في المؤسسات. يدعم التحديث التلقائي للصيانة السلسة.

NSIS MSI MSIX Auto-Updateتحديث تلقائي

How It Works

آلية عمل النظام

1

Upload Photo

رفع صورة المقال

Upload a photo or scanned image of your handwritten essay directly into the desktop app. Supports drag-and-drop and file browser for easy importing.

ارفع صورة أو صورة ممسوحة ضوئياً لمقالك المكتوب بخط اليد مباشرة في تطبيق الحاسوب. يدعم السحب والإفلات ومتصفح الملفات للاستيراد السهل.

2

OCR Extract

استخراج النص بـ OCR

Local OCR engine (Tesseract, PaddleOCR, EasyOCR) or Vision LLM extracts and digitizes your handwritten text. All processing happens on your machine — no data leaves your computer.

محرك OCR محلي (Tesseract, PaddleOCR, EasyOCR) أو LLM بصري يستخرج ويُرقمن النص المكتوب بخط اليد. جميع العمليات تتم على جهازك — لا تخرج أي بيانات من حاسوبك.

3

AI Assessment

التقييم الذكي

Your local LLM (Llama, Mistral, Qwen) evaluates the essay based on course-specific criteria and provides detailed feedback with scores across all assessment dimensions.

يقوم LLM المحلي (Llama, Mistral, Qwen) بتقييم المقال بناءً على معايير المادة المحددة ويقدم تغذية راجعة مفصلة مع درجات عبر جميع أبعاد التقييم.

4

Get Feedback

الحصول على التغذية الراجعة

Review your scores, read detailed feedback for each criterion, and track your progress over time. All results are saved locally in your SQLite database.

راجع درجاتك، واقرأ التغذية الراجعة المفصلة لكل معيار، وتتبع تقدّمك عبر الزمن. جميع النتائج تُحفظ محلياً في قاعدة بيانات SQLite.

Technology Stack

المكدس التقني

Application Framework

إطار العمل

Built with Electron.js for the desktop shell, React for the frontend UI, and Python FastAPI for the backend AI/OCR processing pipeline.

مبني بـ Electron.js لقشرة سطح المكتب، وReact لواجهة المستخدم الأمامية، وPython FastAPI لخط معالجة الذكاء الاصطناعي والتعرف الضوئي في الخلفية.

Electron.js React FastAPI

AI & OCR Engines

محركات الذكاء الاصطناعي والتعرف الضوئي

Local LLMs via Ollama/LM Studio for AI assessment. Multiple OCR engines (Tesseract, PaddleOCR, EasyOCR) and Vision LLMs (LLaVA, MiniCPM-V) for handwriting recognition.

نماذج LLM محلية عبر Ollama/LM Studio للتقييم الذكي. محركات OCR متعددة (Tesseract, PaddleOCR, EasyOCR) ونماذج LLM بصرية (LLaVA, MiniCPM-V) للتعرف على الخط.

Ollama PaddleOCR Tesseract LLaVA

Distribution & Packaging

التوزيع والتغليف

Packaged with electron-builder supporting NSIS installer, MSI, and MSIX formats. Microsoft Store ready for enterprise deployment with auto-update support.

مُغلف بـ electron-builder مع دعم مُثبّت NSIS وMSI وMSIX. جاهز لمتجر مايكروسوفت للنشر المؤسسي مع دعم التحديث التلقائي.

electron-builder NSIS MSI MSIX
