EAP Writing Assistant (iAWE)

مساعد الكتابة الأكاديمية (iAWE)

Intelligent Automated Writing Evaluation
بيئة ذكية لتقييم الكتابة الأكاديمية

A Google Sheets Add-on designed for EAP students. It combines Gemini 2.5 Flash, the New Academic Word List (NAWL), and Conceptual RAG to provide schema-enforced, bilingual feedback across 13 linguistic dimensions—from lexical sophistication to rhetorical stance.

إضافة لجداول بيانات جوجل مخصصة لطلاب اللغة الإنجليزية للأغراض الأكاديمية (EAP). تجمع بين Gemini 2.5، وقائمة الكلمات الأكاديمية الجديدة (NAWL)، والتوليد المعزز بالمفاهيم (RAG) لتقديم تغذية راجعة ثنائية اللغة ومنضبطة هيكلياً عبر 13 بعداً لغوياً—من التعقيد المعجمي إلى الموقف البلاغي.

Google Data Usage

استخدام بيانات جوجل

The EAP Writing Assistant requires access to your Google Sheets to read student text for analysis and write feedback into the dashboard columns. Text is processed ephemerally by the AI model and is not stored by us.

يتطلب مساعد الكتابة الأكاديمية الوصول إلى جداول جوجل لقراءة نصوص الطلاب للتحليل وكتابة التغذية الراجعة في أعمدة لوحة المعلومات. تتم معالجة النص لحظياً بواسطة نموذج الذكاء الاصطناعي ولا يتم تخزينه لدينا.

The EAP Writing Assistant's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

يلتزم استخدام مساعد الكتابة الأكاديمية للمعلومات المستلمة من واجهات برمجة تطبيقات جوجل بـ سياسة بيانات مستخدمي خدمات Google API، بما في ذلك متطلبات الاستخدام المحدود.

System Workflow

آلية العمل

1

Input & Profiling

المدخلات والتحليل المعجمي

Text is validated and profiled against the hardcoded NAWL dataset (O(1) lookup).

يتم التحقق من النص وتحليله معجمياً مقابل قاعدة بيانات NAWL المدمجة.

2

Conceptual RAG

التوليد المعزز (RAG)

Key dimensions trigger semantic searches to retrieve scholarly definitions (Hyland, Swales).

تستدعي الأبعاد الرئيسية بحثاً دلالياً لاسترجاع التعريفات الأكاديمية الموثقة.

3

Dashboard Output

لوحة التغذية الراجعة

AI feedback is parsed into a 13-column dashboard with bilingual support.

يتم تحليل ردود الذكاء الاصطناعي وتوزيعها في لوحة معلومات مكونة من 13 عموداً.

Citation الاقتباس

Mandour, W. (2026). Enhancing EAP Writing through AI-Driven Bilingual Text Analysis: A Schema-Enforced, Corpus-Informed Approach. [Paper ID: 17].

Core Features

الميزات الأساسية

Schema Enforcement

الالتزام الهيكلي (JSON Schema)

Utilizes Schmitt et al.'s (2024) patterns to force the AI into valid JSON, eliminating conversational fluff and hallucinations.

استخدام أنماط هندسة الأوامر لإجبار الذكاء الاصطناعي على مخرجات JSON دقيقة، مما يلغي الهلوسة.

NAWL Integration

قائمة الكلمات الأكاديمية (NAWL)

Deterministically identifies academic vocabulary coverage (92%) using a local, latency-free script integration.

تحديد تغطية المفردات الأكاديمية بدقة باستخدام دمج محلي سريع داخل البرمجية.

Bilingual Scaffolding

سقالات تعليمية ثنائية اللغة

Provides explanatory feedback in Arabic or English, translating complex Applied Linguistics terminology for learners.

توفير تغذية راجعة توضيحية بالعربية أو الإنجليزية، مع ترجمة مصطلحات اللغويات التطبيقية.
