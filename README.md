## About

Dr. Waleed Mandour is a **Senior Instructor** at the **Center for Preparatory Studies, Sultan Qaboos University (SQU), Oman**, and a standing member of both the **E-Learning Committee** and the **Translation Committee** at SQU. He holds an **MA in Applied Linguistics** from Mansoura University, Egypt, and is a **Google for Education Certified Teacher Trainer**.

His research explores the dynamic intersections of Lexicography, Corpus Studies, Discourse Analysis, and their innovative applications in Educational Technology and AI-powered linguistic tools. He is particularly interested in developing data-driven methodologies and open-source tools that enhance our understanding of language, facilitate effective language learning, and revolutionize dictionary design.

## Research Focus

Dr. Mandour's research program is organized around three interconnected areas:

### 1. Computational Lexicography & AI-Powered Dictionary Design
Investigating how computational methods and Artificial Intelligence can revolutionize lexicography. This includes the development of novel frameworks such as the **AI-Model Triangulation Approach (AMTA)** to streamline and enhance bilingual dictionary compilation through systematic integration of corpus linguistics, RAG architectures, and expert validation.

**Keywords:** Lexicography, AI in Applied Linguistics, Bilingual Dictionaries, Computational Linguistics, AMTA

### 2. Corpus-Based Language Analysis & Discourse Studies
Extensively utilizing corpus linguistics to study language in real-world contexts through building and analyzing specialized corpora (e.g., medical abstracts, political speeches, literary works). Research identifies patterns in semantics, phraseology, collocation, and discourse structures across different linguistic and cultural settings.

**Keywords:** Corpus Linguistics, Discourse Analysis, Semantics, Collocation, Language Variation

### 3. AI in Language Learning & Pedagogical Applications
Investigating the integration of Artificial Intelligence tools to enhance language learning and teaching methodologies. This includes developing and evaluating AI-powered tools for textual analysis, sentiment analysis, simplification, and exploring how large language models can support students and researchers in linguistic tasks.

**Keywords:** AI in Education, Language Learning, CALL, Sentiment Analysis, Text Analysis Tools

## Design System

The website follows a consistent design system across all pages:

### Color Palette
| Variable | Value | Usage |
|----------|-------|-------|
| `--primary` | `#0A1C3C` (Navy) | Headers, titles, primary text |
| `--secondary` | `#20B2AA` (Teal) | Accents, links, CTAs |
| `--accent` | `#1A5F7A` (Steel Blue) | Secondary elements |
| `--light` | `#f0f4f7` (Light Gray) | Backgrounds, cards |
| `--dark` | `#333` (Charcoal) | Body text |
| `--white` | `#ffffff` | Card backgrounds |

### Typography
- **English:** 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- **Arabic:** 'Cairo', 'Segoe UI', Tahoma, sans-serif

### Design Patterns
- Card-based layouts with hover effects (`translateY(-5px)`)
- Responsive grid with `auto-fit`/`auto-fill`
- Bilingual content with `.lang-en`/`.lang-ar` class toggling
- RTL support via `body.arabic` class with `direction: rtl`
- Consistent shadows, border-radius (10-12px), and transitions

## Local Development

### Prerequisites
- A modern web browser
- A local web server (e.g., VS Code Live Server, Python's `http.server`, or Node.js `serve`)

### Running Locally

```bash
# Clone the repository
git clone https://github.com/waleedmandour/Homepage.git
cd Homepage

# Option 1: Python
python3 -m http.server 8000

# Option 2: Node.js
npx serve .

# Option 3: VS Code Live Server extension
# Right-click index.html > Open with Live Server
```

Then open `http://localhost:8000` in your browser.

## Deployment

This site is deployed on **GitHub Pages** with a custom domain. The deployment workflow is defined in `.github/workflows/static.yml` and is triggered automatically on pushes to the `main` branch.

### Custom Domain
The custom domain `waleedmandour.org` is configured via:
- `CNAME` file in the repository root
- DNS settings pointing to GitHub Pages
